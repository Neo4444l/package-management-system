================================================================
环境变量配置模板
================================================================

添加用户登录功能时需要配置环境变量。


================================================================
📝 操作步骤
================================================================

1. 在项目根目录创建一个名为 .env 的文件

2. 复制以下内容到 .env 文件中：

================================================================
# Supabase 配置
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
================================================================

3. 替换为您的真实值：
   - your_supabase_project_url → 您的Supabase项目URL
   - your_supabase_anon_key → 您的Supabase匿名密钥


================================================================
🔍 如何获取这些值
================================================================

1. 登录 Supabase 控制台：https://supabase.com

2. 选择您的项目

3. 点击左侧 Settings → API

4. 复制以下两个值：
   - Project URL → 粘贴到 VITE_SUPABASE_URL
   - anon public key → 粘贴到 VITE_SUPABASE_ANON_KEY


================================================================
📋 示例
================================================================

假设您的Supabase配置如下：
- Project URL: https://abcdefghijk.supabase.co
- anon public: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

那么 .env 文件内容应该是：

VITE_SUPABASE_URL=https://abcdefghijk.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...


================================================================
⚠️ 重要提示
================================================================

1. .env 文件不要上传到Git
   ✅ 已在 .gitignore 中配置，会自动忽略

2. 本地和Vercel都需要配置
   - 本地：创建 .env 文件
   - Vercel：在项目设置中添加环境变量

3. 修改环境变量后需要重启开发服务器
   - 停止当前服务器（Ctrl+C）
   - 重新运行 npm run dev


================================================================
🌐 Vercel环境变量配置
================================================================

1. 访问 Vercel 控制台

2. 选择您的项目

3. Settings → Environment Variables

4. 添加两个变量：
   Name: VITE_SUPABASE_URL
   Value: 您的Supabase URL

   Name: VITE_SUPABASE_ANON_KEY
   Value: 您的Supabase Anon Key

5. 点击 Save

6. 重新部署项目
