第2步：配置 .env 文件
----------------------------------------
在项目根目录创建 .env 文件：

VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

从 Supabase 控制台获取这两个值：
Settings → API → Project URL
Settings → API → anon public key
