包裹管理系统 - 快速部署命令
================================================================

【第一步】初始化Git仓库
----------------------------------------------------------------
cd "C:\Users\31424\Desktop\新建文件夹"
git init
git add .
git commit -m "初始提交：包裹管理系统"


【第二步】创建GitHub仓库并上传
----------------------------------------------------------------
1. 访问：https://github.com/new
2. 仓库名：package-management-system
3. 设置为 Public（公开）
4. 不要勾选任何初始化选项
5. 点击 "Create repository"

6. 在本地执行（替换 YOUR_USERNAME 为您的GitHub用户名）：

git remote add origin https://github.com/YOUR_USERNAME/package-management-system.git
git branch -M main
git push -u origin main


【第三步】部署到Vercel
----------------------------------------------------------------
1. 访问：https://vercel.com
2. 点击 "Sign Up" 使用GitHub登录
3. 授权Vercel访问您的仓库
4. 点击 "Add New..." → "Project"
5. 选择 "package-management-system" 仓库
6. 点击 "Import"
7. 直接点击 "Deploy"（无需修改任何配置）
8. 等待1-3分钟部署完成


【第四步】访问您的网站
----------------------------------------------------------------
部署成功后，您会获得一个网址：
https://package-management-system.vercel.app

（实际网址可能略有不同）


【后续更新网站】
----------------------------------------------------------------
每次修改代码后：

git add .
git commit -m "更新说明"
git push

Vercel会自动重新部署！
